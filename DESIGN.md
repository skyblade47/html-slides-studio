# HTML Slides Studio - 设计方案

> 用 HTML + AI 替代 PPT 的演示创作工具

**版本**：v1.0  
**日期**：2026-05-29  
**状态**：初稿

---

## 一、项目定位

**核心目标**：打造一款 AI 驱动的 HTML 演示文稿创作工具，让用户通过自然语言即可生成专业级的演示页面，同时保留类似 PPT 的编辑和演示体验。

**目标用户**：
- 个人知识分享者
- 培训讲师
- 需要轻量化演示工具的商务人士

### 工作流程

1. **创建项目** → 输入主题
2. **AI 生成大纲** → 用户调整
3. **选择/上传审美模板**

## 三、核心模块

| 模块 | 功能描述 |
|------|----------|
| **项目管理器** | 创建、列表、搜索、删除演示项目 |
| **大纲编辑器** | 树形结构组织页面章节，可拖拽排序 |
| **模板系统** | 预设 5 种主题 + 自定义审美 Skill |
| **AI 生成器** | 根据大纲和模板生成完整 HTML |
| **可视化编辑器** | 拖拽编辑文字、图片、图表、布局 |
| **演示播放器** | 全屏、无菜单的 PPT 风格展示 |
| **发布工具** | 打包本地文件 + 一键发布到托管平台 |

## 四、审美 Skill 系统

### 4.1 概念定义

审美 Skill 是一套可复用的设计参数包，用于指导 AI 生成符合特定风格的演示页面。

### 4.2 Skill 结构

```
审美 Skill 包含：
├── 参考图片（截图/Figma）
├── 提取的风格参数
│   ├── 主色调
│   ├── 字体组合
│   ├── 间距规则
│   ├── 动画风格
│   └── 卡片/阴影风格
└── 使用示例
```

### 4.3 工作流程

1. 用户上传参考图片
2. AI 分析提取风格特征
3. 保存为可复用的 Skill
4. 生成时自动匹配最接近的 Skill

### 4.4 内置审美 Skill 库

| 类别 | 风格描述 | 适用场景 |
|------|----------|----------|
| 📊 商务专业 | 深蓝+白底、简洁扁平风、数据图表优先 | 商业汇报、项目展示 |
| 🎓 学术教育 | 清晰层级、图表+公式、温和配色 | 课程讲解、论文答辩 |
| 💡 创意展示 | 大图背景、渐变色彩、动感排版 | 产品发布、创意提案 |
| 🌿 清新简约 | 大量留白、柔和色调、极简图标 | 个人分享、知识记录 |
| 🏢 政府公文 | 正式庄重、国标配色、规范化排版 | 政府汇报、政策宣讲 |

### 4.5 分层设计

1. **系统内置库** — 开箱即用，覆盖常见场景
2. **用户自定义库** — 上传参考图，生成个人 Skill
3. **分享/导入** — 用户可以导出 Skill 给其他人

## 五、技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **桌面框架** | Tauri v2 | 轻量、安全、打包体积小 |
| **前端框架** | React + TypeScript | 组件化开发 |
| **样式方案** | TailwindCSS | 快速构建 UI |
| **编辑器** | Fabric.js | Canvas 拖拽编辑 |
| **AI 集成** | OpenAI / Claude API | 大纲生成、内容生成、风格分析 |
| **本地存储** | SQLite + 文件系统 | 项目元数据 + HTML/素材 |
| **图表库** | ECharts / Chart.js | 数据可视化 |
