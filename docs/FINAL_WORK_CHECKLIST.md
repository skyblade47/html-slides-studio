## 项目概述

### 产品定位
**HTML 演示文稿工作室** - AI 驱动的 HTML 演示文稿生成工具

### 核心特性
- ✅ 单 HTML 文件输出，无需依赖
- ✅ AI 辅助生成演示大纲
- ✅ 多种模板风格选择
- ✅ 丰富的动画效果库
- ✅ 智能动画推荐系统
- ✅ 浏览器直接打开，无需安装

### 技术栈
- Tauri (桌面应用框架)
- React 18 (UI 框架)
- TypeScript (类型安全)
- Vite (构建工具)
- Zustand (状态管理)
- Tailwind CSS (样式框架)

## 不应引入的功能

以下功能**不应引入**（除非用户明确要求）：

| 功能 | 原因 |
|------|------|
| 后端服务器依赖 | 违背"单文件输出"理念 |
| 大型第三方库 | 违背"轻量级"定位 |
| 商业授权组件 | 违背开源免费原则 |
| 可执行文件打包 | 违背"HTML传输"理念 |
| 复杂数据库 | 违背简单性原则 |

## 版本发布流程

### 发布前检查

1. **功能完整性**
   - [ ] 所有计划功能已完成
   - [ ] 无阻塞性 bug
   - [ ] 用户文档已更新

2. **代码质量**
   - [ ] TypeScript 0 错误
   - [ ] 构建成功
   - [ ] 无 console.error
   - [ ] 性能可接受

3. **测试**
   - [ ] 功能测试通过
   - [ ] 边界情况处理
   - [ ] 错误提示友好

4. **文档**
   - [ ] README 更新
   - [ ] CHANGELOG 记录
   - [ ] 错误库更新

### 版本号规范

- **MAJOR.MINOR.PATCH**
- MAJOR: 不兼容的重大更新
- MINOR: 向后兼容的功能添加
- PATCH: 向后兼容的问题修复
